## App Functionalities
1. สามารถเพิ่ม ลบ แก้ไข ชื่อ เบอร์โทรศัพท์ แท็กและสีได้
2. เก็บรายชื่อโดยเรียงตามตัวอักษร (A-Z, a-z)
3. สามารถ Tag ของรายชื่อนั้นๆได้ คือ General, Mobile, Home และ Work
4. สามารถ Tag สีของรายชื่อได้
